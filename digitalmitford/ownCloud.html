Owncloud is the new file management system for the Digital Mitford Project. This document will introduce you to basic features.

Logging In

Navigate to http://digitalmitford.org/owncloud/. The default password for new users is 'mitford.' Once you log in, you should see three main folders and a user manual. Documents and photos are personal folders that you can use to store your own files. You can change your password by going to the top right of the screen where you see you see your username and icon. Click on the dropdown and go into users, where you will be able to edit not only your password but also your icon, which is how you will appear to other users, your email, and how often you recieve email notifications.

Navigating the file system

The home icon in the top left will take you back to the initial screen you came into. As you click into a folder, you will see an expanded path go across the screen, and you can get back to any step in the path by clicking on the word. Try exploring! You can always get back home with one click. This can also be used to move files into parent directories. Simply drag a file into the desired directory along the top!

Adding Files

You can drag files from your desktop and drop them into the correct folder icon or the name of the folder on the path bar.

Deleting Files

You can delete files by clicking the inline trash icon - please do not delete files that are not yours!

Downloading Files

You can download a file with the inline download icon.

Making Comments

Clicking 'comments' inline will allow you to make a comment in the field with a '-' and then clicking save. Your comment will be automatically tagged with your username and the time and date of comment. As far as we are aware, you cannot delete comments so make sure you know what you want to write before you press save!

Versions

OwnCloud tracks versions - you can access a dropdown of the different versions by clicking the circular arrow inline. Restoring a previous version replaces the current version with the selected version, and the version that was open when you clicked 'restore' becomes the most recent saved version.
