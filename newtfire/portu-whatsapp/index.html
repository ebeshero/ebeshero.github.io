Welcome to the PortuWhatsApp Homepage

This is the homepage of the PortuWhatsApp project for the University of Pittsburgh's Computational Methods in the Humanities course. This project analyzes the conversations between native Portuguese speakers and native English Portuguese learners through the texting app "WhatsApp." You can read more about the project and how the data was collected on the About page. The three students that are responsible for working on this project are linguistics majors and, in order to fulfil the Methods requirement for the degree, have undertaken a project that interacts with some linguistic issue or topic.
