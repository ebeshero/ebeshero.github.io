Project Coding Information

The project team has stayed connected and up-to-date with each other's work by maintaining a GitHub repository dedicated to the project. For information about how the project began and to see how the project developed over time, visit the Issues board of our GitHub repository.

To view the Harlem Renaissance Poetry Project's GitHub repository, click here.

The GitHub repository above is where you'll find our coding information -- the foundation of both this website and our research. There, you'll find:

  • Our poems in their individual XML documents, with our markup
  • Our poem collections in a singular XML document
  • Our Relax NG schema
  • Our Schematron schema
  • Our HTML documents that form this website
  • Various XSLT documents that helped us generate our XML documents, HTML documents, and SVG graphs
  • Our SVG documents containing our graphs

The JavaScript and CSS code that powers our click-to-highlight menus on the poetry collection pages was adapted from Elisa Beshero-Bondar's code on the Digital Archives and Pacific Cultures project.
